# Noise Lab

Самодостаточная веб-песочница для процедурной генерации. Один файл
`noise-lab.html` — открывается в любом браузере, работает офлайн, всё
считается локально. Заточена под тач/Android.

## Возможности

- **Шумы:** Value, Perlin, Worley (клеточный)
- **Фрактал (fBm):** октавы, lacunarity, persistence, ridged
- **Domain warp:** искажение координат вторым шумом
- **Стек слоёв** с режимами наложения: Normal, Add, Multiply, Screen,
  Subtract, Max, Min + вес каждого слоя
- **Операции над яркостью слоя:** множитель, сдвиг, гамма, инверсия
- **Геометризация:** выделение областей по порогу → обводка каждой
  прямоугольником/эллипсом по габаритам → заливка фигуры яркостью порога
- **Маска по основе:** слой применяется только там, где яркость нижних
  слоёв попадает в диапазон (с растушёвкой)
- **Отображение:** ч/б карта высот и rogue-тайлы (пещеры через клеточный
  автомат)
- **Импорт/экспорт:**
  - конфигурация (JSON) — сохранить и загрузить обратно весь стек
  - Python-скрипт — `generate(seed) -> 2D-массив яркости 0..1`,
    битово-идентичный движку приложения (только стандартная библиотека)

## Конвейер одного слоя

```
шум → fBm → domain warp → нормализация 0..1 → операции → геометрия
    → (маска применяется при смешивании с нижними слоями) → композит
```

Итоговый композит нормализуется в 0..1 и подаётся в отображение.

## Экспортированный Python

Функция `generate(seed=0, size=None)` возвращает список списков (яркость
0..1). `generate(0)` точно воспроизводит текущую карту из приложения.
Запуск `python3 <файл>.py <seed>` дополнительно сохраняет `noise.pgm`.

```python
from noise_export import generate
grid = generate(42)          # 2D список
# при желании: import numpy as np; arr = np.array(grid)
```
